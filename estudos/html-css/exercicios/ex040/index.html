Media Queries são uma funcionalidade do CSS usada para criar layouts responsivos, adaptando o estilo de uma página web a diferentes dispositivos e tamanhos de tela, como smartphones, tablets, laptops e desktops. A ideia principal por trás das media queries é a de aplicar diferentes estilos com base em determinadas condições, como a largura da viewport, a orientação do dispositivo (retrato ou paisagem), a resolução da tela, entre outras características. Como Funcionam as Media Queries? As media queries consistem em uma regra condicional que verifica características do ambiente de exibição e aplica estilos específicos apenas quando essas condições são atendidas.
